## Installation and Usage
Download it as a [ZIP package](https://github.com/taitokurssiGit/detector/archive/master.zip) or by Git: 
```sh
git clone https://github.com/taitokurssiGit/detector.git
```
Start it:
```sh
python detector.py
```
